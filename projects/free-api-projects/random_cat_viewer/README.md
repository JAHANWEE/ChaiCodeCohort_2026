# Random Cat Viewer

A fun cat breed viewer built with React + Vite + Tailwind CSS.

**Live demo:** https://random-cat-viewer-omega.vercel.app/

## Features

- Fetches a random cat breed on load and on button click
- Displays breed image, name, origin, temperament tags, and lifespan
- Loading spinner and error state handling

## Tech Stack

- React 19, Vite 8, Tailwind CSS 4
- FreeAPI — `https://api.freeapi.app/api/v1/public/cats/cat/random`

## Getting Started

```bash
npm install
npm run dev
```
